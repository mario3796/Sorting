# Sorting

- This is Java Desktop Application, its function is sorting unsorted arrays using 6 types of sorting:
* Insertion Sorting
* Selection Sorting
* Merge Sorting
* Heap Sorting
* Bubble Sorting
* Quick Sorting
- All you should do enter the size of array not more than 10 or less than 0, then fill the element of array, then choose the type of sorting,
then the output will be a sorted array, then you can restart the operation.
